Objetivo:

Realizar o gerenciamento das ordens de serviço da empresa através de um sistema web que foi desenvolvido com html 8, utilizando o framework Laravel. Algumas das funcionalidades do sistema:

  • Vincular

  • Excluir
  • Iniciar
  • Finalizar
  • Pausar
  • monitoramento em tempo real
  • Dentre outros
